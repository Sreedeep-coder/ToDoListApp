# To-Do List App (Java)

## 📌 Project Overview
A simple **To-Do List application** built using **Java Swing**.  
It allows users to add, remove, and clear tasks. This project demonstrates GUI development and event handling in Java.

## ✨ Features
- Add new tasks
- Remove selected tasks
- Clear all tasks
- Interactive GUI

## 🛠️ Tech Stack
- Java
- Swing (for GUI)
